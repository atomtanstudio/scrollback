A self-hosted archive for X/Twitter saves, RSS feeds, articles, media, prompts, and AI-searchable research trails.

Keep the things you meant to come back to.

Demo: https://scrollback.atomtan.studio

## Optional Thread Follow-Ups

1.

I built Scrollback because my saved posts, threads, RSS reads, article links, and prompts kept disappearing into separate tools.

I wanted one place I could search later, enrich when useful, and host myself.

2.

What it does today:

- X/Twitter post, thread, media, and article-card capture
- RSS/Atom ingestion
- Keyword and optional semantic search
- Optional AI summaries, tags, categories, translations, and image descriptions
- Export and read-only remote query APIs

3.

For local setup, Scrollback supports SQLite.

For hosted installs and semantic search, use PostgreSQL or Supabase with pgvector.

4.

For account safety: one-click X capture can work without the X API, but bulk importing Likes and Bookmarks is best done with the official X API bearer token or OAuth flow.

5.

There is a public read-only demo so you can click around before installing it:

https://scrollback.atomtan.studio

Feedback from self-hosters, RSS people, data hoarders, and people with too many saved posts is very welcome.
